# CSVファイルの照合機能

## 目的
新旧システムにより発行した、社員の一ヶ月間の勤怠を集計した2つのCSVファイルの指定の項目の値を照合します。これにより、新システムの導入の判断をしやすくします。
- **項目:**
	{
		"社員ID",
		"勤務形態"
		"実働時間計",
		"リアル実働時間",
		"年休(全日)",
		"年休(半日)",
		"時間外",
		"時間休計"
	}

## 機能要件
- 照合した後、値が違う項目に関しては、json形式で出力します。
	```json
	{
		社員ID: [
			{項目名: {"旧": 値, "新": 値}},
			︙(あれば繰り返し)
		 ],
		社員ID: [
			{項目名: {"旧": 値, "新": 値}},
			︙
		],
		︙
	}
	```

## 実装方針
1. "社員ID"を元に、他の項目を照合してください。
2. 旧システムのCSVファイルは、全社員分。新システムのCSVファイルは、常勤・パートに分かれています。ですので、片方に社員IDがなくても、エラーを出さず、処理を続けらるようにしてください。
3. CSVファイルでなければ、エラーを出してください。また、集計ファイルではない、項目が揃っていないなどの、誤ったCSVファイルが指定された場合も同様にしてください。
